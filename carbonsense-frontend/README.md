# CarbonSense Frontend

React + Vite frontend application for CarbonSense.

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Radix UI
- React Query
- Recharts
- Leaflet

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Type Checking

```bash
npm run check
```
